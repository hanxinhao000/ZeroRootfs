##目前支持功能
- [x] 一键安装自带rootfs(ubuntu)
- [x] 自定义安装Rootfs
- [x] 可自由选择快速启动和调试启动
- [x] SD目录访问
- [x] 支持自定义启动脚本
- [ ] 国际化
- [ ] 模拟x86 Rootfs功能
- [ ] 容器功能支持
- [ ] pulseaudio 支持
- [ ] 更多proot版本工具
      
## 下载自己的ROOTFS
在此网站下载属于你自己的rootfs
https://images.linuxcontainers.org/images
然后放于自己能找到的内部存储目录
点击 开始安装 体验吧
